# DESN 374 × DOL AI Literacy Framework — Alignment Report

A comprehensive web-based alignment report demonstrating how **DESN 374: AI + Design** — a university course offered at Eastern Washington University — aligns with and exceeds the U.S. Department of Labor's AI Literacy Framework (TEN 07-25, released February 13, 2026).

## Overview

This interactive report provides a detailed analysis showing that DESN 374, which has been taught since Fall 2023 and is currently in its 6th offering (Winter 2026), already meets every foundational content area and delivery principle outlined in the federal framework for AI literacy education.

### Key Alignment Results

- **5/5** Foundational Content Areas — Fully Addressed
- **7/7** Delivery Principles — Fully Implemented  
- **6** Areas Exceeding the Framework — Beyond Baseline

## Features

- **Interactive Collapsible Sections** — Expand/collapse individual content areas or all at once
- **Scroll Reveal Animations** — Smooth animations as content enters the viewport
- **Responsive Design** — Optimized for all screen sizes
- **Accessible** — Includes skip links, ARIA attributes, and keyboard navigation
- **Professional Typography** — Uses Newsreader, DM Sans, and JetBrains Mono fonts
- **Modern UI** — Primer Design System-inspired color tokens with dark/light sections

## Technologies Used

- **HTML5** — Semantic markup for content structure
- **CSS3** — Custom properties, grid layouts, animations
- **Vanilla JavaScript** — No dependencies, lightweight interactions
- **IntersectionObserver API** — Efficient scroll-based animations

## Getting Started

### Viewing the Report

Simply open `index.html` in a modern web browser:

```bash
# Option 1: Direct file opening
open index.html  # macOS
xdg-open index.html  # Linux
start index.html  # Windows

# Option 2: Using a local server (recommended)
python -m http.server 8000
# Then navigate to http://localhost:8000
```

### Features & Interactions

- **Expand/Collapse Sections** — Click on any content area header to toggle details
- **Expand/Collapse All** — Use the toggle button at the top of each section
- **Scroll to Top** — Click the button in the bottom-right corner (appears on scroll)
- **Keyboard Navigation** — All interactive elements are keyboard accessible

## Framework Context

The U.S. Department of Labor's AI Literacy Framework (TEN 07-25) establishes a national standard for AI literacy education, defining it as:

> "A foundational set of competencies that enable individuals to use and evaluate AI technologies responsibly, with a primary focus on generative AI, which is increasingly central to the modern workplace."

This report demonstrates DESN 374's comprehensive coverage of:
- What people need to know about AI (foundational content)
- How AI literacy should be taught (delivery principles)
- Areas where the course goes beyond the baseline framework
